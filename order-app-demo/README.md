## Less支持的配置
在React脚手架构架的环境中，默认支持的是Css和sass/scss。所以需要自己配置Less
1. 执行命令:npm run eject(创建完项目不要做任何操作，直接执行次命令),如果我们修改了文件，打开文件根目录，显示隐藏文件，删除.git文件夹，再次执行命令
2. 安装依赖
```js
npm install --save-dev less less0loader
```
